# Stardance Contribution: Fixing cachix/devenv

I contributed to [cachix/devenv](https://github.com/cachix/devenv), a Nix-based tool for creating reproducible developer environments.

I fixed the bug where the devcontainer configuration (which is crucial for codespaces for example) was being generated as a symlink to the nix store.
Nix store is a place where everything generated by nix lives and crucially, it is unique to system, so codespaces broke, since the symlink lead to non-existing file.

It took two attempts to find an elegant solution:

- [#2932](https://github.com/cachix/devenv/pull/2932) Closed
  - Frankly a bit over-engineered, but while discussing this PR, some other person added a long requested feature - copy mode for files.
- [#2951](https://github.com/cachix/devenv/pull/2951) Merged
  - A far simpler solution, implemented thanks to copy mode introducted because of my first PR.
